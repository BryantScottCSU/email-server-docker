# email-server-docker

This project involved deploying and configuring a functional email server on a Linux VPS using Docker.

*Key components*

* Postfix for SMTP mail delivery

* Dovecot for IMAP mailbox access

* OpenDKIM for DKIM signing

* Docker for containerized deployment

*Technical work*

*Configured Docker-based services for mail handling

*Set up DNS records including MX, SPF, DKIM, and PTR

*Verified end-to-end email sending and receiving using a custom domain

Security & privacy

*Configuration files, credentials, and private keys are intentionally not included in this repository.

*The live server is password-protected and not publicly accessible.

*What I learned*

*SMTP vs IMAP responsibilities

*How SPF, DKIM, and PTR work together for mail authentication

*Practical Linux server administration and debugging

*Containerized service deployment
